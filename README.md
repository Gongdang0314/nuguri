## 3. 컴파일 및 실행 방법

### Windows
- 컴파일
    gcc -o nuguri.exe nuguri.c
- 실행
    nuguri.exe

### Linux/macOS
- 컴파일
    gcc -o nuguri nuguri.c
- 실행
    ./nuguri
